# Automatas 1 - Código Python

Este repositorio contiene un código de ejemplo desarrollado para la materia **Automatas 1**. El código implementa un autómata que acepta cadenas de caracteres con los símbolos "a" y "b". El autómata valida las cadenas según las siguientes reglas:

- La cadena debe tener un número impar de "a"s.
- La cadena debe tener un número par de "b"s.

## Alcance del Código

El código implementa un **autómata finito determinista (DFA)** que evalúa si una cadena de caracteres cumple con las condiciones mencionadas. El autómata tiene cuatro estados posibles y acepta una cadena si al finalizar la evaluación la cadena tiene el número adecuado de "a"s y "b"s.
